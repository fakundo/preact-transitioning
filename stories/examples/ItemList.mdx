# Transition Group â€“ Item List Example

This example demonstrates how to use the `TransitionGroup` component with an array of items to manage animations when adding or removing elements.

## Usage

To use the `TransitionGroup` in your project, you can follow the example below:

```tsx
<TransitionGroup>
  {items.map((item) => (
    <FadeTransition key={item.id}>
      <div>{item.content}</div>
    </FadeTransition>
  ))}
</TransitionGroup>
```
